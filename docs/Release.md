# PO,RR

윈도우에서 예배 레파토리와 가사 텍스트를 하나의 파워포인트 파일로 만드는 프로그램입니다.

## 개요

PO,RR는 압축 파일을 푼 뒤 바로 실행할 수 있습니다. 별도 설치 과정은 필요하지 않습니다. 작업 표시줄과 창 제목에는 `PO,RR by a tempo`가 표시됩니다.

- `LyricsToPPT.exe`와 같은 폴더에 있는 `template.pptx`를 사용합니다.
- 실행 창의 `레파토리 입력` 창에 적힌 곡 순서대로 슬라이드를 생성합니다.
- 곡 제목과 같은 이름의 가사 `.txt` 파일을 읽어 파워포인트 파일을 만듭니다.
- 프로그램을 삭제하려면 압축을 푼 폴더 전체를 제거하면 됩니다.

기본 폴더 구성:

```text
LyricsToPPT/
  LyricsToPPT.exe
  template.pptx
  assets/
    atempo.png
    background.png
  sequences_sample.txt
  가사1.txt
  가사2.txt
```

## 다운로드

최신 버전은 [깃허브 배포 페이지](https://github.com/GraceCheong/ppt-gen/releases)에서 다운로드할 수 있습니다.

예시 파일명:

```text
LyricsToPPT-Windows-v1.0.0.zip
```

## 설치

### 압축 파일 풀기

압축 파일을 마우스 오른쪽 버튼으로 클릭한 뒤 `압축 풀기` 또는 `모두 추출`을 선택해 압축을 풀어요.

압축을 푼 뒤 폴더 예시:

```text
LyricsToPPT-Windows-v1.0.0/
  LyricsToPPT.exe
  template.pptx
  assets/
    atempo.png
    background.png
  sequences_sample.txt
  README.md
```

### 템플릿 확인

`template.pptx`는 배포 파일에 포함되어 있습니다. 이 파일은 `LyricsToPPT.exe`와 같은 폴더에 있어야 합니다.

직접 만든 템플릿으로 교체하려면 파일명을 `template.pptx`로 맞춰요.

슬라이드 마스터에는 아래 레이아웃이 필요합니다.

```text
제목
가사
```

## 레파토리와 가사 준비

### 레파토리 입력

실행 창의 `레파토리 입력` 창은 두 줄을 한 곡 단위로 읽습니다.

- 첫 줄: 곡 제목
- 둘째 줄: 파트 순서

예시:

```text
한나의 노래
I-V1-V2-C-Inter-V2-C-C-Out
나의 하나님
I-V1-V1-C-Inter-V2-C-B-C-C
```

같은 폴더에 `sequences.txt`가 있으면 시작할 때 자동으로 불러옵니다. 기존 파일을 계속 쓰고 있었다면 내용을 그대로 `레파토리 입력` 창에 붙여 넣어도 됩니다.

### 가사 파일 작성

`레파토리 입력` 창에 적은 곡 제목과 같은 이름으로 가사 파일을 만들어요.

```text
한나의 노래.txt
```

가사 파일에는 파트 이름과 가사를 순서대로 작성합니다.

```text
V1
가사 첫 번째 줄
가사 두 번째 줄

C
후렴 첫 번째 줄
후렴 두 번째 줄

Out
마지막 가사
```

## 사용 방법

### 프로그램 실행

`LyricsToPPT.exe`를 두 번 클릭한 뒤 `레파토리 입력` 창에 곡 제목과 파트 순서를 입력하고 `레파토리 인식`을 눌러요.

오른쪽 곡 목록에서 곡을 선택하면 가사를 바로 수정할 수 있습니다. 가사 파일이 없거나 비어 있으면 입력 형식 가이드가 표시됩니다.

가사 파일이 없으면 `가사 다운로드` 버튼으로 자동 다운로드를 먼저 시도할 수 있습니다. 수정한 가사는 자동으로 저장되며, 이후 `파워포인트 생성`을 눌러요.

기본 설정:

```text
슬라이드당 줄 수: 2
```

처음 실행할 때 윈도우 보안 경고가 표시되면 `추가 정보` > `실행`을 선택해요.

### 결과 파일 확인

작업이 완료되면 `out` 폴더 안에 아래 파일이 생성됩니다.

```text
out/integrated_lyrics.pptx
```

## 관리

### 프로그램 폴더 이동

폴더 전체를 그대로 옮기면 다른 위치나 다른 PC에서도 사용할 수 있습니다.

### 프로그램 삭제

설치형 프로그램이 아니므로 압축을 푼 폴더를 삭제하면 제거됩니다.

### 버전 업데이트

새 버전을 다운로드한 뒤 기존에 사용하던 파일을 새 폴더로 복사해요.

```text
template.pptx
각 곡 제목.txt
```

## 문제 해결

문제가 발생하면 아래 항목을 열어 확인해요.

<details>
<summary>template.pptx를 찾을 수 없다는 메시지가 표시되는 경우</summary>

- `template.pptx`가 `LyricsToPPT.exe`와 같은 폴더에 있는지 확인해요.
- 직접 만든 템플릿으로 교체했다면 파일명이 정확히 `template.pptx`인지 확인해요.

</details>

<details>
<summary>가사 파일을 찾을 수 없다는 메시지가 표시되는 경우</summary>

- `레파토리 입력` 창의 곡 제목과 가사 파일명이 정확히 같은지 확인해요.
- 띄어쓰기, 특수문자, 한글 표기까지 파일명과 일치시켜요.

예시:

```text
레파토리 입력 창의 곡 제목: 한나의 노래
가사 파일명: 한나의 노래.txt
```

</details>

<details>
<summary>파워포인트 파일 저장에 실패하는 경우</summary>

- `out/integrated_lyrics.pptx`가 파워포인트에서 열려 있으면 닫고 다시 실행해요.
- 이미 같은 이름의 파일이 있으면 기존 파일명을 바꾼 뒤 다시 생성해요.

</details>

<details>
<summary>창은 열리지만 결과 파일이 생성되지 않는 경우</summary>

- `레파토리 입력` 창이 `곡 제목`과 `파트 순서`의 두 줄 단위로 작성되어 있는지 확인해요.

형식:

```text
곡 제목
파트 순서
곡 제목
파트 순서
```

</details>

<details>
<summary>실행 파일만 따로 복사해 실행한 경우</summary>

- `LyricsToPPT.exe`만 실행할 수는 있지만 작업 파일이 없으면 파워포인트 파일을 만들 수 없어요.
- 압축을 푼 폴더 전체를 옮겨 사용해요.

필수 파일:

```text
LyricsToPPT.exe
template.pptx
가사 파일들
```

</details>

<details>
<summary>내가 만든 PPT 템플릿에 적용하고 싶은 경우</summary>

`template.pptx`를 직접 만든 파일로 바꾸면 원하는 디자인을 적용할 수 있어요. 파일명은 반드시 `template.pptx`로 맞추고, `LyricsToPPT.exe`와 같은 폴더에 두어야 합니다.

프로그램은 슬라이드 마스터의 레이아웃 이름을 보고 사용할 슬라이드를 찾습니다.

- 레이아웃 이름에 `제목`이 들어가면 곡 제목 슬라이드로 사용합니다.
- 레이아웃 이름에 `가사`가 들어가면 가사 슬라이드로 사용합니다.
- 해당 이름의 레이아웃이 없으면 기본 순서의 레이아웃을 대신 사용합니다.

곡 제목 슬라이드에는 텍스트를 입력할 수 있는 자리 표시자가 하나 이상 필요해요. 프로그램은 첫 번째 텍스트 자리 표시자에 곡 제목을 넣어요.

가사 슬라이드에는 텍스트 자리 표시자를 두 개 두는 것을 권장해요.

- 가장 큰 텍스트 자리 표시자: 가사 본문
- 두 번째로 큰 텍스트 자리 표시자: 곡 제목

가사 슬라이드를 만들 때는 아래 항목을 확인해요.

- 가사 본문 영역은 슬라이드에서 가장 크게 만듭니다.
- 곡 제목 영역은 가사 영역보다 작게 만듭니다.
- 장식용 텍스트 상자는 자리 표시자가 아닌 일반 텍스트 상자로 두는 것이 좋습니다.
- 글자 크기, 줄 간격, 정렬은 템플릿에서 미리 지정해 둡니다.
- 배경 이미지나 도형은 텍스트 영역을 가리지 않도록 배치합니다.

</details>
